# PythonSWG 🎮🐍💧🔫

A simple command-line Snake-Water-Gun game built with Python. Play against the computer and test your luck!

## 🕹️ How to Play
- `s` → Snake
- `w` → Water
- `g` → Gun

The rules:
- Snake drinks water → Snake wins
- Gun shoots snake → Gun wins
- Water douses gun → Water wins
- Same choice → Draw

## 🚀 Run the Game

```bash
python main.py
